# Cudas Info Alexa Skill

Give info on upcoming softball games for the Cudas.

### build
- `zip -r deploy_pack.zip /path/to/project/lambda/src/*`

### develop
- `pip install {module-name} -t /path/to/project/lambda/src`

More info can be found [here](http://docs.aws.amazon.com/lambda/latest/dg/lambda-python-how-to-create-deployment-package.html).
